# WinFormsLottery

A simple lottery application built with Windows Forms and .NET 8.0.

## Features

- Simple and intuitive user interface
- Random selection functionality
- Cross-platform support with .NET 8.0

## Requirements

- .NET 8.0 SDK or later
- Windows 10/11 with Windows Forms support

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/chjantonychen/WinFormsLottery.git
   ```

2. Open the solution in Visual Studio or VS Code

3. Build and run the application

## Project Structure

- `Program.cs` - Application entry point
- `MainForm.cs` - Main form
- `WinFormsLottery.csproj` - Project file
